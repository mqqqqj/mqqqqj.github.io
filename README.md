# Personal Academic Homepage

This repository hosts a simple static academic homepage for GitHub Pages.

## Files

- `index.html`: homepage content.
- `assets/css/academic-home.css`: page styles.
- `assets/img/prof_pic.jpg`: profile image.
- `404.html`: simple fallback page.
- `.nojekyll`: tells GitHub Pages to publish the files as plain static assets.

No Jekyll, Ruby, Node, or build step is required.

To edit the site, update `index.html` directly and replace `assets/img/prof_pic.jpg` with your own photo when ready.
